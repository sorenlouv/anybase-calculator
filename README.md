## Contribute

Install dependencies:

```
npm install
```

Start server:

```
npm start
```

Run tests:

```
npm test
```

Run linting and formatting:

```
npm run format && npm run lint
```

Deploy to GH Pages:

```
npm run build && npm run deploy
```
